Steps:
Hosted Ubuntu Virtual Machine on Oracle Virtual Box Oracle Virtual Box : https://www.oracle.com/in/virtualization/technologies/vm/downloads/virtualbox-downloads.html Ubuntu VM ISO : https://ubuntu.com/download/desktop
Install Pycharm/ Visual Studio
Setup Python
Clone Git repository https://github.com/Vikas098766/Microservices.git (Modified changes)
Create Virtual Environment (python -m venv venv)
Install dependencies from requirements.txt file (pip install -r requirements.txt)
Docker Build creation (docker build -t my-flask-app .)
Run the containerized application as a prediction service (docker run -p 5000:5000 my-flask-app)
